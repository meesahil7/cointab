## cointab
Sign Up and Login app with JWT...

## Features -->
1. Sign up and login using JWT and bcrypt.
2. Form validation
3. If the user enters wrong password 5
times consecutively, blocks the user for
24 hours from the last incorrect
attempt.
4. Logout

## Packages -->
1. mongoose
2. jsonwebtoken
3. bcrypt
4. formik
5. yup
